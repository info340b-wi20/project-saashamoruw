Project Hub

By: Saasha Mor

Overview

There are a lot of great projects produced by people. Project Hub is a one-stop-shop to show case any projects you worked on whether it was for a class, capstone, a hackathon or even a passion project. Your project can be anything from Software Development and Data Science to Graphic Design and UI/UX Design!
What if you don't have any projects to showcase? Don't worry! Project Hub can also help users find budding projects in need of your skills and time.

Existing Solutions

  • Github brings together the world's largest community of developers to discover, share, and build better software. It has capability to have open source projects to private team ones
  • Open Science Framework is a platform to showcase all your research projects with the ability to upload sources from Github, Dropbox and Amazon S3

What's missing?

The point of showcasing projects are to show them off. It is a great way to show the world your skills while working on a topic that intrigues you as well as add value to the world with a working prototype that people can use. Although, the existing solutions are great to be able to collaborate on projects, they are not good ways to "show-off" projects

Proposed Application

Here is what a user would typically do on Project Hub:

  1. Find Team Members for Projects
    1. Users can browse all the available open projects
    2. Each project will have a topic and requirements for team members
    3. Users can filter projects on the basis of
      • Role of Team Member needed
      • Time commitment of entire project
      • Time commitment per week
      • Purpose of project: Hackathon, Capstone etc.
      • Payment
      • Location
  2. Showcase Projects
    1. Users can add details of the projects they have completed
    2. User adds brief descriptions of the project along with team member roles
    3. Users can add purpose of the project like Hackathon, Capstone, Personal etc.
    4. They can add source code or any other materials from Github or by upload
    5. Projects will require some sort of working proof of the project like website, prototype images etc.
    6. Teammates can also indicate which section of Creative Commons it comes under

Potential Users

  • This web app can be used by anyone ranging from high school kids to professionals in industry
    • benefit from having a project showcase plaform

  • People can also find products that may be of use to them through this web app

How will this help?

This web app provides a platform for students to

  1. Find projects to work on that interest them
  2. Find teammates to complete project ideas
  3. Showcase (with pride) all their projects
